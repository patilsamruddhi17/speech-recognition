# Speech Recognition

A Python project that records voice and converts speech into text using SpeechRecognition and PyAudio.

## Features
- Record audio from microphone
- Convert speech to text
- Save audio temporarily

## Technologies Used
- Python 3.11
- SpeechRecognition
- PyAudio

## Installation

```bash
pip install SpeechRecognition
pip install pyaudio
```

## Run Project

```bash
py -3.11 "speech recog code.py"
```
